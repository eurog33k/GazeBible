# Bible Translation Licenses

Source site: https://www.biblesupersearch.com/bible-downloads/
Site statement: *"The Bibles available on this page are FREE to use and distribute for non-commercial purposes."*

GazeBible is a non-commercial app. To eliminate all copyright risk, only translations with a **confirmed** free-use or public domain status are included. Any translation with an unclear, missing, or restricted license has been removed.
